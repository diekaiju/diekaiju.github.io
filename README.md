# Vide Coder Landing Page

This is a cyberpunk-themed landing page designed for a "Vide Coder".

## Features
- **Glitch Effect Text**: Animated ease-in/out glitch text.
- **Custom Cursor**: Interactive cursor with hover states.
- **Profile Layout**: Specialized container for the profile image with scanline and overlay effects.
- **Responsive Design**: Adapts to mobile screens.

## How to View
Simply open `index.html` in your web browser.

## Customization
- **Profile Image**: Replace `profile.png` with your own image (keep the filename or update `index.html`).
- **Text**: Edit `index.html` to change the "VIDE CODER" title or the "CREATIVE DESTRUCTION" headers.
- **Colors**: Edit the `:root` variables in `style.css` to change the color scheme.
